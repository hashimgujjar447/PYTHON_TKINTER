## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hashimgujjar447/PYTHON_TKINTER.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PYTHON_TKINTER
   ```
3. Run the desired project:
   ```bash
   python <project-file>.py
   ```
